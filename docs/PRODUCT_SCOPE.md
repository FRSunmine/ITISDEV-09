# SideQuest MVP scope

## Product statement
SideQuest is a campus-focused micro-freelancing platform connecting verified student freelancers with student organizations, university departments, startups, and small businesses needing short-term services.

## Core outcomes
- Students discover quests, apply, manage work, submit deliverables, receive feedback, and build verified portfolio entries.
- Clients create structured quests, define binary deliverables, compare applicants, monitor work, and approve outputs.
- Administrators verify users, review reports, and monitor operational analytics.

## In scope for the current frontend milestone
- Demo role-based login routing
- Student verification screen
- Student dashboard and quest marketplace
- Quest details drawer and application modal
- Student and client quest workspaces
- Student profile and verified portfolio
- Client dashboard, create-quest wizard, and applicant matrix
- Admin operations and analytics
- Static mock data and simulated UI actions

## Explicitly out of scope
- Real authentication or authorization
- Backend/database persistence
- Payment processing or escrow
- AI matching, RAG, or MCP actions
- Blockchain or cryptographic credentials
- Mobile-native application
- Real-time chat, sockets, file storage, or notifications
- Multi-university deployment
